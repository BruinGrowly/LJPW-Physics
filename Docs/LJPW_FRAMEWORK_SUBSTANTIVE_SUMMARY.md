#### Mapping Methodology

**Three Coordinate Assignment Methods**:

1. **Interpolation**: Derive coordinates by shifting from anchor concepts
   ```python
   new_coords = anchor_coords + [L_shift, J_shift, P_shift, W_shift]
   ```

2. **Blending**: Weighted combination of two anchor concepts
   ```python
   new_coords = weight₁ × anchor₁ + weight₂ × anchor₂
   ```

3. **Direct Assignment**: Explicit coordinate placement for distinct concepts
   ```python
   new_coords = [L, J, P, W]  # Empirically determined
   ```
