| Phase | L - P | Characteristic |
|-------|-------|----------------|
| Rise | Positive (+0.20) | Growth, expansion |
| Peak | Near zero (-0.02) | Maximum stability |

## The Pattern

```
L > P  →  System grows (investing in cohesion returns Power)
L ≈ P  →  System stable (balanced expenditure)
L < P  →  System declines (Power consumption exceeds regeneration)
```

## Significance

Traditional metrics track **absolute Power** (military strength, GDP, territory). LJPW reveals the **relative L/P ratio** is the stability indicator.

A system can have enormous Power and still collapse if Love is too far behind.

## Application

When assessing organizations:
- Track L and P independently
- Calculate differential over time
- Negative trending differential = early warning signal
